Desarrollo de sitios web UNAD 2021

Propósito

  • Diseñar y Desarrollar los elementos para construir el sitio web utilizando HTML5 dando estilos con CSS3
  • El propósito de esta página es desarrollar las actividades propuestas en la fase 3. Donde se desarrollaran temas del uso de CSS como el flexbox y el Grid Layout.
  • Demostrar las habilidades adquiridas en el uso de flexbox y css grid layout, por medio del desarrollo de esta actividad.

Introduccíon

El desarrollo de la presente actividad es realizar un sitio web con html 5 utilizando un framework css, para esta actividad se utiliza bootstrap version 5 beta 3,para el desarrollo de la actividad se profundiza en los temas de grid CSS y Flebox por ser el tema escogido.

Se continua con el desarrollo de las diferentes páginas, terminando con la parte de actividades y complementando los demas temas tratados en los temas anteriores del curso.
